Metal3

The Metal3 (pronounced “Metal-kube”) project is an open-source project that provides a set of tools for managing bare-metal infrastructure using Kubernetes.

Get Started

Why Metal3.io?

There are a number of great open source tools for bare metal host provisioning, including Ironic. Metal3.io aims to build on these technologies to provide a Kubernetes native API for managing bare metal hosts via a provisioning stack that is also running on Kubernetes. We believe that Kubernetes Native Infrastructure, or managing your infrastructure just like your applications, is a powerful next step in the evolution of infrastructure management.

The Metal3.io project is also building integration with the Kubernetes cluster-api project, allowing Metal3.io to be used as an infrastructure backend for Machine objects from the Cluster API.
These components integrate seamlessly to leverage the Kubernetes ecosystem and automate the provisioning and management of bare-metal infrastructure.

Cluster API Provider Metal3

Enables the creation and management of physical servers using the Kubernetes Cluster API. It provides a way to provision and manage bare-metal infrastructure using Kubernetes APIs.

Bare Metal Operator

Automates the provisioning of bare-metal servers using the open-source Ironic project. Provides an automated way to deploy and manage bare-metal infrastructure using the Ironic project.

Hardware Manager

Effficiently discovers and manages the hardware resources of bare-metal servers, including power management, BIOS settings, and firmware updates. Provides a way to manage the hardware resources of bare-metal servers.

IP Manager

Provides a way to manage the networking resources of bare-metal servers by managing the IP address allocation and networking configuration of bare-metal servers, enabling the automatic configuration of network interfaces and VLANs

Ironic Image

Images that provide the necessary software to deploy and manage bare-metal infrastructure using the Ironic project, contains the necessary software to run an Ironic deployment.

Ironic

Contains the Ironic conductor and API server, which enable the provisioning and management of bare-metal servers. These images provide the Ironic project's core functionality, enabling the automation and scaling of bare-metal infrastructure.
